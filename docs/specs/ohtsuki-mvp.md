# Ohtsuki MVP 仕様

## 名称

- プロダクト名: Ohtsuki
- 表示名: UFO画像AI判定チェッカー β

## 目的

UFO画像がAI生成・AI加工画像である可能性を確認するためのMVP。
初期版では、判定フローと結果表示の体験を先に整え、実際のAI判定API接続は次フェーズで扱う。

## 初期ページ

- `/ohtsuki`: Ohtsuki のメインページ
- `/ufo-image-checker`: 旧URLからの互換導線

## MVPで扱う判定観点

- AI生成画像の可能性
- AI加工・合成の可能性
- メタデータ確認
- 画像内の不自然な境界、影、反射
- 判定不能なケースの明示

## 初期スコープ外

- 確定的な真贋判定
- UFOそのものの実在判定
- 外部AI APIへの接続
- アカウント機能
- 画像の永続保存
