# Hynek - UFOファンタイプ診断 仕様

## 1. 目的

Hynek は、UFO・宇宙人・アブダクション・コンタクティについての考え方から、ユーザーの「UFO観測スタイル」を楽しく診断する参加型コンテンツである。

アンケートではなく、自己理解と参加体験を重視する。診断結果の表示と、全体傾向の見え方を確認できるようにする。

## 2. 対象範囲

- `/hynek` の単独ページ
- `/hynek/dashboard` のダッシュボードページ
- 1問1画面の診断フロー
- Q11 での分岐に応じた Q12 / Q13 の表示切り替え
- 結果画面
- 全体傾向表示
- 匿名ライブ集計によるダッシュボード表示
- UFO目撃タブでのUFOを見た県のマップ風表示とUFOタイプ集計
- シェア画像風カード

## 3. 非対象範囲

- バックエンド保存
- Supabase / Neon / Postgres 連携
- 投稿連携

## 4. 画面構成

1. 開始画面
2. Q1 から Q15 までの単問画面
3. 属性入力 3 画面
4. 結果画面
5. 日本のUFO観ダッシュボード

ダッシュボードでは、主要サマリーを円形で表示し、宇宙人観タブは「宇宙人観」、正体タブは「UFOってなに？」として案内する。

ダッシュボードのフィルターは、`目撃した人` で目撃回答のみを再集計し、`年代で見る` と `地域で見る` はプルダウンで選んだ条件に応じて再集計する。

Q12 / Q13 は、Q11 で「はっきりある」または「たぶんある」を選んだ場合のみ表示する。

## 5. 結果タイプ

- ロマン派UFOファン
- 証拠重視UFOファン
- 慎重派UFOファン
- 目撃体験重視派
- 不思議好きUFOファン
- ニュース追跡派
- エンタメUFOファン
- 接触・メッセージ関心派

各選択肢には 1 つまたは複数のタイプ加点を設定し、最高得点のタイプを結果とする。同点時は、目撃体験 -> 接触・メッセージ -> ニュース追跡 -> 証拠重視 -> 不思議好き -> 慎重派 -> ロマン派 -> エンタメ の順で固定する。

## 6. 表示内容

結果画面では、以下を表示する。

- あなたのUFOウォッチャータイプ
- 短い解説
- 回答傾向
- 「あなたは全体の◯%に近いです」
- 全体傾向バー
- シェア画像風カード
- 「みんなのUFO観を見る」導線
- 日本のUFO観ダッシュボードでは、タイプ別のランキングを「タイプ別のランキングです。」として案内する。
- 日本のUFO観ダッシュボードでは、`目撃した人` を目撃回答の再集計、`年代で見る` と `地域で見る` をプルダウン選択の再集計として扱う。

結果カードには、タイプごとに用意された男女別の静的アートワークを使用する。性別が「回答しない」の場合は、既定表示を使う。

## 7. デザイン方針

- スマホ優先
- 1問1画面
- 進捗バーを常時表示
- 戻る / 次へを明確に配置
- 占いサイトのような装飾は避ける
- 軽い遊びは選択肢文言にとどめる
- 既存ブランドの落ち着いたトーンに合わせる

## 8. データ取り扱い

- 回答はクライアント state のみで保持する
- 診断結果の送信時に匿名 cookie を使って初回回答のみ集計する
- 再診断は可能だが、同じ端末の再送信は初回のみ集計に反映する
- `/hynek/dashboard` は匿名ライブ集計を表示する

## 9. ディスクレイマー

- 任意回答による非科学的集計である
- UFO・宇宙人・アブダクションの事実認定ではない
- 診断はエンタメと自己理解を目的とする
- 目撃情報は回答者本人の認識に基づく
